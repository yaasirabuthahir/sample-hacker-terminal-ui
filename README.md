# 😈 Fake Hacker Terminal UI

A visually immersive hacker-style terminal interface that simulates command-line interactions. This project recreates a futuristic terminal experience with dynamic command handling and animated outputs.

💎 Part of my 50+ Projects Challenge

---

## ✨ Features
- 💻 Realistic terminal UI design
- ⌨️ Command-based input system
- ⚡ Dynamic output responses
- 🧠 Custom commands (help, about, hack, clear)
- 🔥 Simulated hacking animation
- 📱 Responsive layout

---

## 🛠️ Built With
- HTML5
- CSS3 (Terminal Styling)
- JavaScript (DOM Manipulation & Events)
